# SteadyStride (Anti Tremor Band)

An innovative wearable device designed to stabilize hand tremors for patients with Parkinson's or Essential Tremor.

## Technologies
- **Hardware**: Arduino / ESP32
- **Software**: C++, Embedded Logic
- **Sensors**: Accelerometers, Gyroscopes
